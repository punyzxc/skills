# 🎨 Профессиональные HTML-Презентации

Система для создания красивых, интерактивных и полностью адаптивных HTML-презентаций.

## 📋 Быстрый старт

1. **Скопируйте шаблон** из `template/presentation-template.html`
2. **Отредактируйте основные данные**:
   - Заголовок страницы (в `<title>`)
   - Название презентации (в `<h1>`)
   - Авторов (в `.authors-grid`)
   - Организацию (в `.group-badge` и `.author-footer`)

3. **Добавляйте слайды** как новые `<section>` элементы
4. **Откройте в браузере** и наслаждайтесь!

## 🎯 Ключевые особенности

### ✅ Полностью адаптивные
- Автоматическое масштабирование шрифтов (clamp)
- Отзывчивый дизайн для всех экранов
- Оптимизированные сетки (auto-fit, minmax)

### ✅ Интуитивная навигация
- **Клавиатура**: ↑ ↓ ← → (и Пробел для вниз)
- **Мышь**: прокрутка колеса
- **Сенсор**: свайп вверх/вниз
- **Точки**: клик по навигационным точкам справа

### ✅ Профессиональный дизайн
- Glassmorphism эффект
- Градиентные акценты
- Гладкие анимации
- Темная тема (доступна смена в CSS переменных)

### ✅ Производительность
- Нет зависимостей
- Чистый CSS и JavaScript
- Оптимизированные анимации
- ~50KB минифицированный файл

## 🧩 Доступные компоненты

### 1. Badge (Значок)
```html
<div class="badge animate-on-view">
    <span class="badge-dot"></span>
    Текст значка
</div>
```

### 2. Glass Card (Карточка)
```html
<div class="glass-card accent">
    <div class="icon-circle">
        <svg>...</svg>
    </div>
    <h4>Заголовок</h4>
    <p>Текст</p>
</div>
```

### 3. Сетки
```html
<!-- 2 колонки -->
<div class="grid-2">
    <div>Левая</div>
    <div>Правая</div>
</div>

<!-- 3 колонки -->
<div class="grid-3">...</div>

<!-- 4 колонки -->
<div class="grid-4">...</div>
```

### 4. Списки
```html
<ul class="styled-list">
    <li>Пункт 1</li>
    <li>Пункт 2</li>
</ul>
```

### 5. Карточка статистики
```html
<div class="stat-card">
    <div class="stat-number">100%</div>
    <div class="stat-label">Оригинальности</div>
</div>
```

### 6. Процесс (Flow)
```html
<div class="process-flow">
    <div class="process-step">
        <div class="step-icon">1</div>
        <div class="step-label">Этап 1</div>
    </div>
    <div class="process-arrow">→</div>
    <!-- ... -->
</div>
```

### 7. Ключевой момент
```html
<div class="key-point">
    <div class="key-point-icon">💡</div>
    <div class="key-point-content">
        <h4>Совет</h4>
        <p>Текст совета</p>
    </div>
</div>
```

### 8. Карточка автора
```html
<div class="author-card">
    <div class="author-avatar">А</div>
    <div class="author-name">Имя Фамилия</div>
</div>
```

## 🎨 Персонализация

### Изменение цветовой схемы
Отредактируйте переменные в `:root`:

```css
:root {
    --accent-1: #6366f1;    /* Основной цвет */
    --accent-2: #3b82f6;    /* Второстепенный */
    --accent-3: #0ea5e9;    /* Акцент 1 */
    --accent-4: #06b6d4;    /* Акцент 2 */
}
```

### Добавление своей темы
```css
/* Светлая тема */
:root {
    --bg-dark: #ffffff;
    --text-primary: #000000;
    --text-secondary: #666666;
}
```

## 🚀 Советы для профессиональных презентаций

### 1. Структура
- **Слайд 1**: Титульный (название, авторы, организация)
- **Слайды 2-N**: Контент
- **Последний слайд**: "Спасибо за внимание"

### 2. Лучшие практики
- ✅ Используйте максимум 5-6 пунктов на слайде
- ✅ Добавляйте иконки в SVG для визуальности
- ✅ Используйте `grid-2` или `grid-3` для раскладки
- ✅ Одна идея = один слайд
- ✅ Тестируйте на разных размерах экранов

### 3. Текст
- ✅ Используйте краткие, понятные фразы
- ✅ Избегайте слишком много текста на одном слайде
- ✅ Заголовки должны быть информативны
- ✅ Поддерживайте высокий контраст текста и фона

### 4. Изображения и иконки
```html
<!-- SVG иконка -->
<div class="icon-circle">
    <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <path d="..."></path>
    </svg>
</div>

<!-- Используйте Feather Icons или Font Awesome -->
```

### 5. Анимации
Все элементы с классом `animate-on-view` автоматически анимируются при прокрутке.

Для каскадной анимации используйте `stagger`:
```html
<div class="authors-grid stagger">
    <div class="animate-on-view">...</div>
    <div class="animate-on-view">...</div>
</div>
```

## 🐛 Решение проблем

### Текст не читаем на мобильном
- Проверьте, что используете `clamp()` для размеров
- Увеличьте line-height: минимум 1.6 для текста

### Не работает навигация
- Убедитесь, что JavaScript загружен
- Проверьте консоль браузера на ошибки
- Убедитесь, что все `<section>` имеют `id`

### Слайды прерываются посередине
- Добавьте `min-height: 100vh` к section
- Используйте `overflow: hidden` в section
